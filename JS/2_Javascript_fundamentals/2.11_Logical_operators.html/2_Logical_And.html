&& (AND)

The AND operator is represented with two ampersands &&:

AND “&&” finds the first falsy value

result = value1 && value2 && value3;

The AND && operator does the following: Evaluates operands from left to right. For each operand, converts it to a boolean. If the result is false, stops and returns the original value of that operand. If all operands have been evaluated (i.e. all were truthy), returns the last operand. In other words, AND returns the first falsy value or the last value if none were found.

Precedence of AND && is higher than OR ||

So the code a && b || c && d is essentially the same as if the && expressions were in parentheses: (a && b) || (c && d).
